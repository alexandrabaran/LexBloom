# LexBloom

Simple web app to learn English vocabulary using flashcards.

## Features
- Random word selection
- Text-to-Speech pronunciation
- Offline usage
- No backend

## Data
- Word frequency generated using `wordfreq`
- Translations generated automatically
- IPA generated programmatically
- IPA symbols are part of the International Phonetic Alphabet and are used here for educational purposes.
